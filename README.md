# Naboo Case Study

## Overview
This repository contains a full-stack case study with:

- `back-end`: NestJS GraphQL API with MongoDB (Mongoose)
- `front-end`: Next.js (Pages Router) app with Apollo Client and Mantine

## Tech Stack
### Backend
- NestJS
- GraphQL
- MongoDB + Mongoose
- JWT authentication

### Frontend
- Next.js (Pages Router)
- Apollo Client
- Mantine UI
- Axios
- Vitest

## Run Locally
### Backend
```bash
cd back-end
npm install
npm run start:dev
```

### Frontend
```bash
cd front-end
npm install
npm run dev
```

### Regenerate GraphQL types
```bash
cd front-end
npm run generate-types
```
